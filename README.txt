
Npgsql - .Net Data Provider for PostgreSQL



COMPILATION
===========

	Currently npgsql uses the SharpDevelop IDE for coding and compilation.
It will be updated shortly to an automated compilation suite like NAnt.


INSTALL
=======

	Just copy the resulting .dll (npgsql.dll) to the application directory.


	See RELEASENOTES for more information on this and earlier releases.
